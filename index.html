Projects

Cryptanalysis of the RSA cryptosystem.

Implemented an RSA cryptosystem and investigate the security of it by using different factorization method and compare the efficiency of different methods. The factorization methods used are Trial division, Fermat's factorization, Pollard's p-1, Pollard's Rho, ECM(GMP-ecm)and Quadractic sieve (Msieve). Users can encrypt and decrypt a message using the RSA cryptosystem implemented. This project is made with .NET framework and C++ with the help of some open software.
